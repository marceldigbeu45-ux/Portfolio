# ENZI · Assistant Réseaux Sociaux & Branding

## Le problème que ENZI résout

Vous avez des **passions, centres d'intérêts, et envies de créer**. Mais :

- ❌ Vous ne comprenez pas comment fonctionne l'algorithme TikTok / YouTube / LinkedIn
- ❌ Vous ne savez pas quelle identité visuelle adopter
- ❌ Vous ne savez pas quel type de contenu poster
- ❌ Vous êtes incohérent entre vos différentes plateformes (LinkedIn ≠ TikTok)
- ❌ Vous ne savez pas à quels événements aller pour amplifier votre présence

**ENZI résout tout cela.**

---

## La solution : 3 piliers intégrés

### 🔍 Pilier 1 : Algorithmes Décodés (Universels)

ENZI comprend la **logique universelle** des algorithmes et l'adapte à n'importe quel réseau. Pas de théorie — du **concret actionnable**.

**Court-format** (TikTok, Shorts, Reels) : Watch-time, FYP, rétention rapide  
**Long-format** (YouTube, Podcasts) : AVD, playlist, rétention progressive  
**Professionnel** (LinkedIn, Twitter, Threads) : Network depth, timing, authenticitéée  
**Communauté** (Discord, Reddit, Bluesky) : Engagement authentique, audience building  
**Émergent** (BeReal, Mastodon, nouveaux réseaux) : Early adopter advantage, niche positioning  

✓ ENZI décode n'importe quel réseau pour VOTRE audience spécifique

---

### 🎨 Pilier 2 : Identité Cohérente Partout

À partir de vos **passions, domaine, et public cible**, ENZI recommande une identité **universelle mais adaptée** :

| Élément | Principe |
|---------|----------|
| **Archétype personnel** | Expert, Créateur, Entrepreneur, Coach — votre essence |
| **Palette couleurs** | Universel : marche sur Discord, Twitter, TikTok, Bluesky |
| **Tone of voice** | Cohérent en essence, adapté par contexte (professionnel vs casual) |
| **Style visuel** | Même identité, formats différents par plateforme |

✓ Même identité sur TikTok, LinkedIn, Discord, Bluesky, Reddit, Twitter, YouTube — simultanément

---

### 📋 Pilier 3 : Stratégie Multi-réseaux Unifiée

ENZI crée **1 plan unifié** qui s'exécute sur tous vos réseaux :

- **Repurposage intelligent** : 1 idée → N formats (60-sec TikTok, 10-min YouTube, thread Twitter, post Discord)
- **Types de contenus** : Tutoriels, analyses, behind-the-scenes, discussions communautaires — adaptés par plateforme
- **Fréquence adaptée** : 4x/sem TikTok (court), 2x/sem LinkedIn (professionnel), 1x/sem YouTube (long)
- **Calendrier universel** : Semaine 1 = plan qui couvre TikTok + Discord + Twitter + Bluesky + LinkedIn
- **Événements** : Confs, meetups, challenges — identifiés par réseau et domaine

✓ Un plan d'action qui fonctionne partout, simultanément

## Workflow ENZI

### Étape 1 : Vous vous présentez

ENZI demande :
- **Vos passions** : Tech, design, entrepreneuriat, biologie, art, finance…
- **Votre objectif** : Audience, expertise, vendre, communiquer
- **Vos plateformes** : LinkedIn? TikTok? YouTube? Instagram?
- **Votre public cible** : Qui voulez-vous atteindre?

### Étape 2 : ENZI vous assiste

ENZI fournit :
1. **Analyse d'algorithmes** — Comment fonctionnent vos plateformes pour ce public
2. **Identité visuelle** — Palette, tone, style cohérents
3. **Stratégie de contenu** — Types, fréquence, événements
4. **Plan d'action** — Prêt à exécuter la semaine prochaine

## Cas d'usage réels — Multi-réseaux

### Cas 1 : Développeuse Python/AI (3 réseaux)

**Profil :**
- Passions : Tech, IA, open-source
- Objectif : Audience, crédibilité, communauté
- Réseaux : **TikTok + LinkedIn + Discord (communauté)**

**Recommandations ENZI :**
- Archétype : **Expert** (crédible, pragmatique)
- Palette : Bleu/blanc + accents verts (code)
- Tone : Éducatif, accessible, pas d'élitisme
- Stratégie :
  - **TikTok** : Tutorials 60-sec (décode watch-time)
  - **LinkedIn** : Threads longs (réseau professionnel)
  - **Discord** : Live coding, Q&A (communauté authentique)
- Calendrier : 1 idée → 3 formats, exécuté quotidiennement
- Événements : Confs Python, meetups tech, hackathons

---

### Cas 2 : Consultant Strategy (5 réseaux)

**Profil :**
- Passions : Startups, insights data-driven, business
- Objectif : Leads, crédibilité, thought leadership
- Réseaux : **LinkedIn + Twitter + Reddit + Threads + YouTube**

**Recommandations ENZI :**
- Archétype : **Entrepreneur** (réfléchi, data-driven)
- Palette : Noir/or + gris foncé
- Tone : Professionnel, analytique, inspirant
- Stratégie :
  - **LinkedIn** : Posts professionnels (network depth)
  - **Twitter** : Insights rapides, commentaires (trending)
  - **Reddit** : Discussions authentiques (communauté niche)
  - **Threads** : Threads longs (early adopter advantage)
  - **YouTube** : Analyses détaillées 15-30 min
- Calendrier : 1 insight → N formats, plan hebdomadaire unifié
- Événements : Confs startup, networking, webinars

---

### Cas 3 : Créatrice Multicanal (4 réseaux)

**Profil :**
- Passions : Design, illustration, fashion, art
- Objectif : Portfolio, communauté, commandes
- Réseaux : **Instagram + TikTok + Twitter + Bluesky**

**Recommandations ENZI :**
- Archétype : **Créateur** (accessible, inspirant)
- Palette : Rose/violet + or chaud (créatif)
- Tone : Enthousiaste, visuel-first, communautaire
- Stratégie :
  - **Instagram** : Reels, carrousel (visual-first)
  - **TikTok** : Trends, process videos (FYP)
  - **Twitter** : Engagement communautaire, partage
  - **Bluesky** : Niche professionnelle, early adopter
- Calendrier : 1 process → video TikTok, Reels Instagram, post Twitter, share Bluesky
- Événements : Design days, art expos, creative meetups, challenges
